# Thư mục CV

Đặt **CV của bạn** vào đây (1 file duy nhất). Hỗ trợ: `.pdf`, `.docx`, `.txt`.

Ví dụ: `cv/cv_cua_toi.pdf`

- Công cụ sẽ **tự động** lấy file đầu tiên trong thư mục này làm CV chuẩn để chấm độ phù hợp.
- Hoặc chỉ định rõ qua biến môi trường `CV_PATH` trong `.env`, hoặc tham số `--cv`.
- File CV thật **không được commit lên git** (đã cấu hình trong `.gitignore`) để bảo vệ thông tin cá nhân.
